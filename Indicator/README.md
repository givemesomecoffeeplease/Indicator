# Indicator

교회 워십팀을 위한 라이브 공연 섹션 인디케이터. Logic Pro의 마커와 재생 위치를 실시간으로 읽어 팀원들의 iPad/폰 브라우저에 현재 섹션을 표시합니다.

## 기능

- Logic Pro 마커 목록에서 곡/섹션 자동 파싱
- MTC(MIDI Timecode)로 실시간 재생 위치 수신
- 다음 섹션 전환까지 비트 카운트다운 (4→3→2→1)
- 섹션 진행률 바 실시간 표시
- 로컬 웹서버(포트 8888)로 브라우저에 SSE 푸시
- macOS 메뉴바 앱

## 마커 규칙

Logic Pro 마커 목록에서:
- **곡 마커**: `#곡이름` (예: `#은혜`) — 예비박 시작 위치에 배치
- **섹션 마커**: 일반 텍스트 (예: `Intro`, `Verse`, `Chorus`)

## 설정

### Logic Pro MTC 설정
1. Logic Pro → 파일 → 프로젝트 설정 → 동기화 → MIDI 탭
2. 대상 드롭다운에서 `Logic Pro 가상 출력` 또는 `IAC Driver Bus 1` 선택
3. MTC 체크박스 활성화

### IAC Driver 활성화 (선택)
1. macOS → 오디오 MIDI 설정 → 윈도우 → MIDI 스튜디오
2. IAC Driver 더블클릭 → "기기가 온라인 상태임" 체크

### 마커 목록 창
Logic Pro에서 **탐색 → 마커 목록 열기** 로 창을 열어두어야 앱이 마커를 읽을 수 있습니다.

## 접속

앱 실행 후 메뉴바 아이콘 클릭 → IP 주소 확인 → 같은 WiFi의 기기에서 브라우저로 접속

```
http://[맥IP]:8888
```

## 빌드

- Xcode 16+
- macOS 15+
- 접근성 권한 필요 (시스템 설정 → 개인 정보 보호 → 손쉬운 사용)
- 앱 샌드박스 비활성화 (ENABLE_APP_SANDBOX = NO)
